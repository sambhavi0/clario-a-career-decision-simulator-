# 🚀 Clario-Career Decision Simulator

An interactive web application that helps users discover the most suitable career paths based on their skills, interests, and goals.

---

## 🎯 Overview

Choosing the right career can be confusing and overwhelming.  
This project simplifies that process by providing personalized career recommendations along with clear, actionable roadmaps.

Users input their:
- Skills  
- Interests  
- Goals  
- Experience level  

And the system generates the top 3 career paths best suited for them.

---

## ✨ Features

### 🎯 Personalized Recommendations
- Suggests top 3 career options tailored to the user
- Avoids generic suggestions using intelligent filtering

### 🧠 Smart Matching Algorithm
- Uses weighted logic:
  - Interests (highest priority)
  - Skills
  - Goals
- Filters out irrelevant careers for accurate results

### 🧭 Career Roadmaps
Each recommendation includes:
- Difficulty level  
- Estimated time to achieve the goal  
- Market demand  
- Step by step guidance  

### 🎨 Interactive UI
- Multi step form for better user experience  
- Fully responsive design  
- Smooth and intuitive navigation  

### 🌍 Diverse Career Options
- 30+ career paths across:
  - Technology  
  - Finance  
  - Marketing  
  - Design  
  - Humanities  

### ⚡ Fast & Lightweight
- Built with pure JavaScript  
- No heavy APIs required  
- Instant results

## 🧠 How It Works

1. User selects skills, interests, and goals  
2. System analyzes inputs using a scoring algorithm  
3. Careers are ranked based on:
   - Matching interests  
   - Relevant skills  
   - Aligned goals  
4. Top 3 careers are displayed with detailed guidance
